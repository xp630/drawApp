# iPad 儿童画图 app 设计文档

## 1. 项目概述

- **项目名称**: DrawApp
- **项目类型**: iPad 儿童绘画应用
- **目标用户**: 9 岁儿童
- **核心功能（第一阶段）**: 基础画布绘画，暂不包含 AI 功能
- **后续规划**: AI 上色、AI 画布补全

## 2. UI/UX 设计

### 2.1 页面结构

- **单页面应用** — 画布 + 可折叠工具栏，无多页面导航

### 2.2 布局架构

```
┌────────┬────────────────────────────────────────────┐
│        │                                            │
│  工具栏  │                                            │
│ (可折叠) │              画布区域                      │
│        │           (尽可能最大化)                    │
│  颜色   │                                            │
│ [调色盘] │                                            │
│        │                                            │
│  画笔   │                                            │
│ [细][中][粗] │                                        │
│        │                                            │
│ [橡皮]  │                                            │
│ [清空]  │                                            │
│        │                                            │
│ [草稿箱]│ ← 点击弹出草稿列表                         │
│ [关闭]  │                                            │
└────────┴────────────────────────────────────────────┘
```

**草稿箱弹窗:**
```
┌─────────────────────────────┐
│  草稿箱                  [X] │
├─────────────────────────────┤
│  ┌────┐  ┌────┐  ┌────┐   │
│  │图1 │  │图2 │  │图3 │   │
│  └────┘  └────┘  └────┘   │
│  3月24日  3月23日  3月22日  │
│                            │
│  [+ 新建]                  │
└─────────────────────────────┘
```

- 工具栏位于左侧，可收起以最大化画布区域
- 工具栏宽度: 约 200pt
- 工具栏可折叠，点击关闭按钮收起

### 2.3 颜色方案

- 背景色: `#FFFFFF`（纯白）
- 工具栏背景: `#F5F5F5`（浅灰）
- 主色调: 暂不设定，聚焦黑白灰阶原型

### 2.4 交互设计

| 交互 | 行为 |
|------|------|
| 手指触摸画布 | 绘制线条 |
| 点击调色盘 | 弹出颜色选择器 |
| 点击画笔粗细 | 切换细/中/粗 |
| 点击橡皮擦 | 切换橡皮擦模式 |
| 点击清空 | 清空画布（带确认提示） |
| 点击草稿箱 | 弹出草稿箱列表 |
| 点击关闭 | 收起工具栏 |

## 3. 功能设计

### 3.1 核心功能

| 功能 | 描述 |
|------|------|
| 自由绘画 | 手指在画布上绘制线条，线条跟随手指移动 |
| 颜色选择 | 通过调色盘选择任意颜色 |
| 画笔粗细 | 三档可选: 细(3pt)、中(8pt)、粗(15pt) |
| 橡皮擦 | 擦除已绘制的线条 |
| 清空画布 | 一键清除所有绘制内容 |
| 草稿箱 | 自动保存作品 + 作品列表管理 |

### 3.2 草稿箱功能

| 功能 | 描述 |
|------|------|
| 自动保存 | 每30秒自动保存当前画布到草稿 |
| 草稿列表 | 以缩略图网格展示所有草稿 |
| 恢复草稿 | 点击草稿可恢复到画布继续编辑 |
| 删除草稿 | 支持删除不需要的草稿 |
| 新建画布 | 从草稿箱可新建空白画布 |

### 3.3 技术实现

- **框架**: SwiftUI
- **绘画引擎**: Canvas View + Touch Events
- **调色盘**: ColorPicker（SwiftUI 内置）
- **数据持久化**: UserDefaults（草稿元数据）+ FileManager（画布图片）

## 5. 后续规划（AI 功能）

- **AI 上色**: 选中线稿，调用云端 API 进行智能上色
- **AI 补全**: 画一半，AI 推断并补全剩余部分
- **技术方案**: Python FastAPI 本地服务 + OpenAI/Stability AI API
